## üíª Usage

### Basic Configuration

1. **Target URL**: Enter the URL of the website you have permission to test
2. **Number of Requests**: Set the total number of requests to send
3. **Parallel Threads**: Configure how many requests run simultaneously
4. **Request Method**: Choose between GET or POST requests
5. **Email** (Optional): Receive a summary report via email
6. **Proxy List** (Optional): Upload a .txt file with proxy addresses

### Running a Test

1. Fill in all required fields in the form
2. Click "Start Analysis"
3. Monitor real-time progress and statistics
4. Review results in the analytics dashboard
5. Check the completion popup for final statistics

## üìä Understanding Results

- **Total Requests**: Number of requests successfully sent
- **Success Rate**: Percentage of successful responses (HTTP 2xx-3xx)
- **Average Response**: Mean response time in milliseconds
- **Response Time Chart**: Visual representation of performance over time
